### 数据结构基础

#### **课程信息：**  
- 学分：2.5
- 教师：**陈翔**老师
- 分数构成：
  - 平时分 60% （作业 10% + 小测 10% + 期中 15% + Projects 25%/30% + Bonus 4%）

  - 期末 40%（40分是斩杀线，可覆盖期中成绩）

#### Project

- 分hard/和normal，开学会有一次测试，hard平时分会有五分的加分，相应的工作量和难度会大一些（点名表扬自动微分器）

- 采取互评机制

- 不要拖ddl

报告非常推荐使用[mem的模板](https://www.cc98.org/topic/5840106)，比起一次就要编译很久的Latex，Typst更加轻量化，语法类似markdown，如果熟悉前者语法的话上手起来很快

#### Bonus

-  一共2道编程题构成，难度没有特别大，不过每次都会考察算法的时间复杂度（~~比如我每次都有一个点运行超时~~）

#### 期中

正常复习，每个班貌似都不太一样，也不会安排统一的考试，因为有bonus政策，所以能考80分左右平时分基本就能满了

#### **期末**

**题型**：10 道判断 + 20 道单选 + 2 道程序填空 + 1 道函数题

历年卷在98和Github很多，就不一一列举了。

个人复习策略：主要策略是先过一遍知识点，然后面向历年卷复习（善用搜索），不懂的地方就看笔记。
